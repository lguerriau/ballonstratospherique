Le dossier projet LoRa contiens tout le code qui permet d'envoyer les données de télémétrie sous forme de trame APRS Weather (en utilisant RadioLib) a partir de la RaspberryPi.
Celui-ci a besoin des autres dossiers projet BME280, LM75 et (enevntuellement) MPU6050 pour pouvoir être executé.

- BME280 contiens le code servant a récolter les données de télémétrie du capteur BME280 (Température Externe, Pression Atmosphérique, Humidité)
- LM75 contiens le code servant a récolter les données de télémétrie du capteur LM75 (Température Interne)
